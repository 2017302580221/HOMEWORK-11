2. 第二题

   + 题目

     ![p10](pic\p10.jpg)

   + 解答

     + a

       A平均吞吐量 = PA(1 - PB )

       总体效率 = PA(1 - PB) + PB(1 - PA)

     + b

       代入公式：

       A平均吞吐量 = PA(1-PB) = 2PB - 2(PB^2)

       B平均吞吐量 = PB(1-PA) = PB - 2(PB^2) 

       由以上知A平均吞吐量不比B大的两倍；

       若要使其成立则有 PA(1-PB) = 2PB(1-PA)

       得到：PA = 2PB/(PB+1)

     + c

       A平均吞吐量 = 2P((1-P)^(N-1))

       其他任意节点平均吞吐量 = P(1-2P)((1-P)^(N-2))
